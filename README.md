# Programação Web II – Módulo V

Repositório destinado às atividades, exercícios e projetos da disciplina **Programação Web II**, cursada no **5º módulo** do curso de **Análise e Desenvolvimento de Software**.

## 📚 Sobre a Disciplina

A matéria aborda o desenvolvimento de aplicações web com foco em práticas modernas de programação relevantes para o desenvolvimento full stack.

## Tecnologias e Ferramentas Utilizadas:

- Back-end: Node.JS { utilizando os frameworks: Express / fastfy / nest.js }
- Front-end: React.JS
- Integração com APIs, rotas, autenticação, 
- Banco de dados { Mysql / Postgree }
- Docker
- Microserviços
- Git 
- Deploy

## 📘 Aula 1 – Criação de Servidor e Refatoração

- Introdução a tecnologias utilizadas, incluindo o uso de **frameworks opinativos** com Node.js.
- Utilização da biblioteca **Nodemon** para escuta ativa e reinício automático do servidor.
- Configuração inicial do projeto com o **`package.json`**.
- Construção básica de um servidor com um **endpoint padrão**.
- **Refatoração do projeto**: separação do código em arquivos distintos, com a criação do arquivo `server.js` para isolar o processo de escuta (`listen`) e deixar o `app.js` responsável apenas pela configuração da aplicação.
